## Top Recommendations (Start Here)

The highest-earning actors on the Apify Store typically share these traits:

1. **They solve a painful, recurring business problem** (e.g., lead lists, competitor monitoring, price tracking)
2. **They output structured, ready-to-use data** (CSV/JSON/Excel) that plugs into CRMs or analytics tools
3. **They use Pay Per Event (PPE)** pricing tied to the value of each output (per result, per lead, per page)
4. **They are reliable** - handle anti-bot protection, retries, and edge cases gracefully
5. **They have clear, SEO-optimized READMEs** so people find them via Google and the Apify Store search

## How to Use This Directory

1. **Pick a category** that matches your skills or interests.
2. **Read the detailed idea file** for specs, input/output schemas, and monetization notes.
3. **Copy the `actors/example-scraper` template** in this monorepo as a starting point.
4. **Build the Actor**, test it locally with `apify run`, then push with `apify push`.
5. **Write a great README** (see the marketing checklist in `README.md`).
6. **Price it with PPE** and publish to the Apify Store.
7. **Promote it** on Reddit (r/webscraping, r/SEO), Product Hunt, Twitter/X, and LinkedIn.
